# Tourist Guide App

A multi-screen iOS application developed using **Swift** and **UIKit** to help tourists discover attractions, restaurants, hotels, and cultural places in one convenient mobile application.

The project was designed and developed from scratch, covering the entire development process from UI/UX prototyping to application implementation.


## Project Overview

The Tourist Guide App is designed to simplify travel planning by providing users with categorized information about tourist attractions, restaurants, hotels, and cultural locations.

Before development, the application was planned using Figma prototypes to define navigation, user flows, and screen layouts. The project focuses on creating an intuitive user experience through clear navigation and reusable interface components.


## Features

* Browse tourist attractions
* Explore restaurants and hotels
* View detailed information for each location
* Category-based navigation
* Multi-screen application
* User-friendly interface
* Custom navigation flow
* Responsive Auto Layout


## UI/UX Design

The application interface was designed in **Figma** before implementation.

The design process included:

* User flow planning
* Wireframes
* High-fidelity mockups
* Navigation design
* Screen layout planning

Figma prototypes are available in the **figma-visual/** folder.


## Application Architecture

The application follows the **MVC architecture** and consists of multiple interconnected screens.

Main modules include:

* Home
* Hotels
* Restaurants
* Resorts
* Detail Pages

Navigation is implemented using **UITabBarController**, **UINavigationController**, **UITableView**, and **UICollectionView**.
